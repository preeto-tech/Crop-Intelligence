Design a modern AgriTech SaaS dashboard UI for a platform called "Crop Intelligence".

Style Direction:
- Premium, clean, modern
- Soft glassmorphism cards
- Light green gradient branding
- Organic rounded shapes
- Minimal clutter
- Data-focused layout
- Subtle shadows
- Professional but farmer-friendly look

Layout Requirements:

1. Left Sidebar Navigation:
- Dashboard
- Crop Library
- Community
- Mandi Prices
- Weather
- Transport Booking
Use simple outline icons, modern spacing, active state highlight in green gradient.

2. Top Navbar:
- Language switcher (English / Hindi / Tamil)
- Notification bell
- Profile avatar
- Search bar

3. Main Dashboard Content:
- Welcome header with farmer name
- Weather summary card (temperature + condition icon)
- Mandi price highlights (top 3 crops)
- Recent community posts preview
- Active transport booking status

4. Cards Design:
- Rounded 16-20px corners
- Soft glass background (white with low opacity)
- Shadow depth for elevation
- Clean typography (Inter / Poppins style)

5. Color Palette:
- Primary: Green gradient (#16a34a to #22c55e)
- Background: Light off-white (#f8fafc)
- Accent: Earthy brown for agriculture theme
- Text: Dark slate

6. Responsive design for desktop and tablet.

The overall design should look like a modern startup SaaS product, but adapted for farmers.
